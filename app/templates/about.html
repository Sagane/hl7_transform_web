HL7 Transform

A web tool that allows to transform HL7 messages using a mapping scheme

This tool uses hl7_transform library, developed at Doctolib, and is available under MIT license. The source code can be downloaded in Github. This tool lives from contribution, so if you find a bug or want to improve the application, feel free to fork the code and make a pull request.

Even though we are avid users of the HL7 v2 message format, the authors or the tool itself are not affiliated with HL7 oeganization.
